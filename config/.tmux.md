# Tmux Reference

## Purpose

Terminal multiplexer for persistent sessions and window management.

## Basic Usage

```
tmux                    start new session
tmux new -s <name>      named session
tmux ls                 list sessions
tmux attach             attach to last session
tmux attach -t <name>   attach to named session
tmux kill-session -t <name>
```

## Prefix Key

All commands start with: `Ctrl+A`

## Sessions

```
Ctrl+A d        detach (session keeps running)
Ctrl+A $        rename session
Ctrl+A s        list sessions
```

## Windows

```
Ctrl+A c        new window
Ctrl+A ,        rename window
Ctrl+A n        next window
Ctrl+A p        previous window
Ctrl+A 0-9      jump to window number
Ctrl+A &        kill window (with confirmation)

Alt+Left        previous window (no prefix)
Alt+Right       next window (no prefix)
```

## Panes (splits)

```
Ctrl+A |        vertical split
Ctrl+A -        horizontal split

Ctrl+A h        move left
Ctrl+A j        move down
Ctrl+A k        move up
Ctrl+A l        move right

Ctrl+A H        resize left
Ctrl+A J        resize down
Ctrl+A K        resize up
Ctrl+A L        resize right

Ctrl+A x        kill pane (no confirmation)
Ctrl+A z        zoom pane (fullscreen toggle)
```

## Copy Mode

```
Ctrl+A [        enter copy mode
  v             start selection
  y             copy and exit
  q             exit without copying

Ctrl+A P        paste
```

Mouse selection auto-copies.

## Other

```
Ctrl+A r        reload config
Ctrl+A ?        show all keybindings
Ctrl+A Ctrl+F   fzf menu
```

## Plugins

Install plugins: `Ctrl+A I` (capital I)
Update plugins: `Ctrl+A U`

### Resurrect & Continuum
Auto-saves sessions every 15 minutes.
Sessions restore automatically after reboot.

```
Ctrl+A Ctrl+S   save manually
Ctrl+A Ctrl+R   restore manually
```

## Workflow Examples

### Local
```
tmux new -s work
# do work...
Ctrl+A d
# close terminal
tmux attach -s work
```

### Remote (SSH)
```
ssh server
tmux new -s deploy
# start deployment
# connection drops
ssh server
tmux attach -s deploy
# deployment still running
```

### Multi-window monitoring
```
tmux new -s monitor
# Window 1: app logs
tail -f app.log

Ctrl+A c
# Window 2: nginx logs
tail -f nginx.log

Ctrl+A c
# Window 3: system monitor
htop

# Switch windows with Alt+Left/Right
```

## Config Location

~/.tmux.conf

## Tips

- Always name your sessions
- Use tmux on remote servers
- Mouse is enabled (click, scroll, select)
- Zoom (Ctrl+A z) for focus
