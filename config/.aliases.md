# Shell Reference

Quick reference for aliases, functions, and hotkeys.

## File Operations

```
ls              eza with icons and directories first
ll              detailed list with git status
lt              tree view (2 levels)
cat             bat with syntax highlighting
less            bat with paging
```

## Navigation

```
z <name>        smart cd (zoxide - learns frequently used paths)
zi              interactive directory selection
```

## Git (Oh-My-Zsh git plugin)

Use built-in git plugin aliases. Most common:

```
gst             git status
ga              git add
gaa             git add --all
gcmsg           git commit -m "message"
gp              git push
gl              git pull
gd              git diff
glog            git log --oneline --graph --decorate
```

Full list: https://github.com/ohmyzsh/ohmyzsh/tree/master/plugins/git

## System

```
top             btop (modern system monitor)
```

**Note:** Use `rg` directly instead of `grep` for regex search.

## Tmux

```
t               tmux
ta              tmux attach
tn <name>       tmux new -s <name>
tl              tmux ls
tk <name>       tmux kill-session -t <name>
```

## Custom Functions

### fe
Find and edit file using fzf with preview.
```
fe
```

### fcd
Find file and cd to its directory.
```
fcd [query]
```

### fkill
Kill process interactively with fzf.
```
fkill [signal]
```
Default signal: 9 (SIGKILL)

### fssh
Connect to SSH host from ~/.ssh/config using fzf.
```
fssh
```

### venv
Activate Python virtual environment from ~/VSC.
```
venv
```

## FZF Integration

### Keybindings
```
Ctrl+R          search command history
Ctrl+T          find files in current directory (with preview)
Alt+C           cd to directory
```

### In commands
Add `**<Tab>` to trigger fzf completion:
```
vim **<Tab>     select file
cd **<Tab>      select directory
kill **<Tab>    select process
ssh **<Tab>     select host
```

### Tab completion
Press Tab for fzf-enhanced completion with preview.
Use `,` and `.` to switch between completion groups.

## SSH Workflow

Add custom SSH shortcuts to ~/.zshrc:
```bash
alias prod='ssh user@server.com -t "tmux attach || tmux new -s prod"'
```

This automatically starts/attaches tmux on remote server.
