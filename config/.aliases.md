# Shell Reference

Quick reference for aliases, functions, and hotkeys.

## File Operations

```
ls              native ls command (unaliased)
l               eza with icons and directories first
ll              eza detailed list with git status
lt              eza tree view (2 levels)
cat             bat with syntax highlighting
less            bat with paging
```

## Navigation

```
z <name>        smart cd (zoxide - learns frequently used paths)
zi              interactive directory selection
```

## Git (Oh-My-Zsh git plugin)

```
gst             git status
```

## Custom Functions

### fe
Find and edit file using fzf with preview.
```
fe
```

### fcd
Find file and cd to its directory.
```
fcd [query]
```

### fkill
Kill process interactively with fzf.
```
fkill [signal]
```
Default signal: 9 (SIGKILL)

### fssh
Connect to SSH host from ~/.ssh/config using fzf.
```
fssh
```

### venv
Activate Python virtual environment from ~/VSC.
```
venv
```

## FZF Integration

### Keybindings
```
Ctrl+R          search command history
Ctrl+T          find files in current directory (with preview)
Alt+C           cd to directory
```

### In commands
Add `**<Tab>` to trigger fzf completion:
```
vim **<Tab>     select file
cd **<Tab>      select directory
kill **<Tab>    select process
ssh **<Tab>     select host
```

### Tab completion
Press Tab for fzf-enhanced completion with preview.
Use `,` and `.` to switch between completion groups.

## Kitty Terminal Shortcuts (macOS)

### Text Navigation
```
Cmd+Left/Right          beginning/end of line
Alt+Left/Right          move by word
Cmd+Backspace           delete to beginning of line
Alt+Backspace           delete word backward
```

### Splits & Windows
```
Cmd+D                   vertical split
Cmd+Shift+D             horizontal split
Cmd+H/J/K/L             navigate splits (vim-style)
Cmd+Shift+Z             zoom/unzoom split
F5/F6                   quick hsplit/vsplit
```

### Tabs
```
Cmd+T                   new tab
Cmd+W                   close window
Cmd+Shift+W             close OS window
Cmd+1-9                 jump to tab number
Cmd+Shift+Left/Right    previous/next tab
```

### Utilities
```
Cmd+Shift+E             open URLs/paths (hints kitten)
Cmd+Shift+S             SSH with fzf selector
Cmd+K                   clear terminal
Cmd+F                   search in scrollback
Cmd+Shift+F5            reload config
```

## SSH Workflow

Add custom SSH shortcuts to ~/.zshrc:
```bash
alias prod='ssh user@server.com -t "tmux attach || tmux new -s prod"'
```

This automatically starts/attaches tmux on remote server.
